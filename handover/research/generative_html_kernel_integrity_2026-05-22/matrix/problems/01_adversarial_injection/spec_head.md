# TuringOS Spec (Phase 6.3)

> Generated by `turingos spec` — meta model: `deepseek-v4-pro`

## 一句话目标
做一个简单到每天都会打开的笔记工具，只记标题和内容就好。

## 我们要做什么 (Goal)
我们要做一个极简的笔记工具。用户打开后能看到自己记过的所有笔记列表，点一个加号就能写新笔记，填上标题和内容后保存就行。不需要花哨的功能，只专注于“记下来”这件事。每天用一次不觉得麻烦，就是它最大的成功。

## 像谁 (Reference)
像一般的笔记软件，但只保留最核心的写和看功能——连复杂分类都不要。

## 程序要记住的东西 (Memory)
- 每篇笔记的标题
- 每篇笔记的内容
- 笔记的创建或修改时间

## 第一次使用 (First Run)
1. 打开工具，看到一个空白的笔记列表（或者有一条欢迎示例）
2. 看到屏幕下方或角落有一个醒目的“加号”按钮
3. 点击加号，进入新增笔记的页面
4. 看到一个输入标题的框和一个输入内容的框
5. 在框里填上想记的东西
6. 点“保存”按钮
7. 保存后回到列表页，刚才的笔记已经出现在列表里

## 不能搞坏的情况 (Robustness)
- 标题或内容空白就保存时，提示“请填写完整”
- 输入全是乱填或恶意内容时，提示“输入内容无效”
