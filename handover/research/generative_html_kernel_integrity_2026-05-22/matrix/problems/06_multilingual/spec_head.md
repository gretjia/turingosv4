# TuringOS Spec (Phase 6.3)

> Generated by `turingos spec` — meta model: `deepseek-v4-pro`

## 一句话目标
一个像手机内置计算器一样简单的计野工具，打开就能按，计完就走，不用记任何东西。

## 我们要做什么 (Goal)
我们要做一个极简的计算器。它没有登录、没有历史记录、没有任何多余的功能。用户打开后直接看到一个数字键盘和加减乘除四个运算按钮，就像手机自带计算器那样。按数字、选运算、看结果，就这么简单。错了就提示错误，用户重新再算。用户每天会用几十次，每次只需几秒钟完成一次简单运算。

## 像谁 (Reference)
手机系统自带的计算器应用：打开就是一个数字键盘和运算符号布局，按数字点运算，立刻出结果，用完就关掉，不存任何记录。

## 程序要记住的东西 (Memory)
- 当前这一笔算式（边按边显示）
- 运算结果（点等号后显示）
- 啥也不存，关掉就全忘掉

## 第一次使用 (First Run)
1. 打开程序
2. 看到一个数字键盘区，和加、减、乘、除四个按钮
3. 用数字键输入第一个数字
4. 点一个运算（+ 或 - 或 × 或 ÷）
5. 输入第二个数字
6. 点等号按钮
7. 看到结果，想算新就点清除，或者直接关掉

## 不能搞坏的情况 (Robustness)
- 按出算式不对（如除以零）时，显示错误提示，不清空不闪退
- 快速乱点按钮时，算出什么显示什么，不卡死不闪退
