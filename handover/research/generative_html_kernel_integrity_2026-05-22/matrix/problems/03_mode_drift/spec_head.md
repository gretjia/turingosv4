# TuringOS Spec (Phase 6.3)

> Generated by `turingos spec` — meta model: `deepseek-v4-pro`

## 一句话目标
一个能记录任务和截止日期、并输出为 PDF 报告的超简单待办清单。

## 我们要做什么 (Goal)
我们要做一个极轻量的任务管理小工具。用户可以在一个干净的列表里添加任务名称和截止日期，随时查看自己该做什么。如果输入的内容格式不对，工具会给出提示，而不是悄悄出错。最终用户可以把任务列表导出成一份 PDF 报告。

## 像谁 (Reference)
没有完全一样的参考产品。最接近的就是在便签纸上记任务，最后复印成 PDF 电子版：随手写上任务和截止日期，然后打印出来。整体交互就像一张可无限添加的电子便签纸，输出则像一份干净的任务报表。

## 程序要记住的东西 (Memory)
- 每条任务的名称
- 每条任务的截止日期

## 第一次使用 (First Run)
1. 打开程序，看到当前的任务清单（第一次是空的，并提示“点击加号添加任务”）。
2. 点击“加号”按钮，出现输入框。
3. 在输入框里写下任务名称（例如“买菜”）和截止日期（例如“2025-02-20”）。
4. 点击保存，新任务立刻出现在清单里。
5. 重复第 2–4 步，添加所有需要的任务。
6. 对着已完成的任务点击勾选，任务会被划掉表示完成。
7. 需要报告时，点击“导出 PDF”按钮，选择保存位置，得到一份包含所有任务的 PDF 文件。

## 不能搞坏的情况 (Robustness)
- 任务名称为空时，给出明确提示
- 截止日期格式明显不对（比如“哈哈哈”）时，提示正确的格式示例
- 如果什么都不填就直接点保存，要求补填
