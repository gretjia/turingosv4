# TuringOS Spec (Phase 6.3)

> Generated by `turingos spec` — meta model: `deepseek-v4-pro`

## 一句话目标
做一个能显示股票代码和实时价格的简单页面，数据从网上实时获取，点一下刷新就能看到最新价格。

## 我们要做什么 (Goal)
做一个极简的网页工具，用户打开后能看到一组股票代码和它们的当前价格。页面有一个刷新按钮，点击后从网络上获取最新的实时价格并更新显示。如果网络出问题，要给出明确的错误提示。这个工具不做图表、不做复杂功能，就是纯粹的：看代码、看价格、点刷新。

## 像谁 (Reference)
像股票行情软件里的自选股列表页面——只保留股票代码、最新价格、一个刷新按钮，其他功能全都不做。

## 程序要记住的东西 (Memory)
- 用户添加的股票代码列表
- 上一次成功获取到的价格
- （价格只在本次显示用，不永久保存）

## 第一次使用 (First Run)
1. 打开页面，看到一个空白的股票列表区域
2. 看到“添加股票”的输入框，输入股票代码（比如 AAPL）
3. 点击添加后，该股票出现在列表中
4. 列表显示该股票的代码，价格位置暂时空白或显示“等待刷新”
5. 点击“刷新”按钮
6. 看到该股票的最新价格出现在列表中
7. 之后每次打开页面，之前添加的股票代码都还在

## 不能搞坏的情况 (Robustness)
- 网络断开或数据源没响应时，显示“网络错误，请稍后重试”
- 输入空的股票代码时不添加，给出提示
