# TuringOS Spec (Phase 6.3)

> Generated by `turingos spec` — meta model: `deepseek-v4-pro`

## One-line Goal
A single self-contained webpage that lets me compare three project launch plans side-by-side, score them, and see which one wins.

## What We're Building (Goal)
A simple browser tool that shows three launch plans in columns. I can give each plan a score from 1 to 5 across several criteria, and the page automatically calculates weighted totals and highlights the best plan. Everything stays in the page while I'm using it — no accounts, no saving, no internet needed after the page loads.

## Like What (Reference)
A lightweight decision matrix app — the kind where you list options, rate them, and get a recommendation — but stripped down to a single file with no sign-ups, no databases, and no external services.

## What the Program Remembers (Memory)
- The three plan names I type in
- The criteria names and their importance weights
- Every score I enter (1 to 5)
- The calculated weighted totals
- Which plan is currently recommended
- All of this only while the page is open

## First Run (First Click Walk)
1. I open the page in my browser
2. I see three columns, each representing a launch plan
3. I type a name at the top of each column
4. I see rows of criteria down the left side
5. I click into a score box and type a number from 1 to 5
6. The weighted total updates immediately at the bottom of each column
7. One plan is clearly marked as the recommended choice

## What It Must Not Break On (Robustness)
- Blank or empty score boxes should not cause errors
- Typing letters or symbols into a score box should be handled gracefully
- All three plans having equal totals should still show a clear result
- Very long plan names should not break the layout
- The page should work without any internet connection after loading

## Deliberately NOT Doing (Out of Scope)
- No login or user accounts
- No backend server or database
- No cloud sync or saving between sessions
- No collaboration or sharing features
- No payment processing
- No external packages, libraries, or CDN dependencies

## Success Looks Like (Acceptance)
- I can paste a launch scenario into the page and compare options in under five minutes
- After a month of use, I can explain my final choice to someone else using what the tool shows me
- The tool works entirely offline with no broken features

## Given/When/Then Examples
- **Given** I have three plans with different scores, **when** I change one score from 3 to 5, **then** the weighted total for that plan updates immediately and the recommendation may change.
- **Given** I leave a score box empty, **when** the totals are calculated, **then** that empty box is treated as zero and the page does not show an error.
- **Given** I type "hello" into a score box, **when** I click away, **then** the box clears or ignores the invalid input and the totals remain stable.
- **Given** all three plans end up with identical weighted totals, **when** I look at the recommendation area, **then** it tells me the plans are tied rather than picking one arbitrarily.
- **Given** I type a very long plan name, **when** the page renders, **then** the name wraps or truncates without pushing other columns off-screen.

## One-line Brief to the AI Coder
Build a single self-contained HTML file with a three-column decision matrix, editable scores from 1 to 5, weighted criteria, automatic totals, a clear recommendation, and zero external dependencies.
