# Phase 7 Real-LLM E2E — Spec Grill Transcript

Persona: 30-something Chinese-speaking hobbyist, tech-comfortable but not a developer. Wants to play a Tetris game in browser.

## Q1 / 8

**Question (read off page):**
> 先不用想程序怎么做。能跟我说说你最近遇到了什么事，让你觉得『要是有个小工具就好了』？比如『我妈每周要算一次社区团购账，Excel 太麻烦』。你的故事是什么？

**My answer (composed live, in-character):**
最近工作有点闷，每次想摸鱼休息一下都得打开手机翻半天找游戏。我以前可喜欢玩俄罗斯方块了，可现在浏览器里找的那些都要登录、广告一堆、还有的得下载。我就想，要是有个网页打开就能玩、不用注册、关掉就走的那种简简单单的俄罗斯方块就好了。

## Q2 / 8

**Question:**
> 有没有哪个网站 / App / 小工具，跟你想要的『有点像』？不用一模一样，一两个相似的地方就行。（如果想不出来：那纸笔 / Excel / 微信群里现在是怎么做的？）

**My answer:**
最像的就是小时候在小霸王游戏机上玩的那个俄罗斯方块——10 格宽、20 格高、七种形状、键盘左右下旋转。网上那些 NES 模拟器网站玩的也是那个版本。我想要的就是那个感觉，但是不用模拟器、不用 ROM，直接网页就跑。

## Q3 / 8

**Question:**
> 想象关掉电脑明天再打开，这个工具应该还『记得』哪些东西？比如团购账本会记得：每个人的名字、买了什么、付了多少、还欠多少。你的工具要记得什么？

**My answer:**
其实就一件事——最高分。我玩完关掉浏览器，明天再打开，能看见上次的最高分留在那儿，激励我打破纪录就行。别的什么用户名、皮肤、设置都不用记。

## Q4 / 8

**Question:**
> 假设我是你的用户，第一次打开这个工具——我看到什么？然后我点什么？然后呢？一步一步告诉我，直到我完成一件事。

**My answer:**
打开网页就直接看到一个黑底的方块格子（左边是 10x20 的游戏区），旁边显示『分数 0』和『最高分 0』，还有一行小字写着『按空格开始』。我按下空格键，第一个方块就从顶部开始往下掉。我用左右箭头平移、上箭头旋转、下箭头加速下落。消行了分数会涨，连消会涨得更多。一直到方块堆到顶部就 Game Over，屏幕上显示『游戏结束 · 按空格再来一次』，最高分如果被打破了就更新。

## Q5 / 8

**Question:**
> 如果有个奇怪的用户，故意乱点乱填——比如把『金额』填成『哈哈哈』，或者同一个名字录入 50 遍——你希望工具怎么办？报错？忽略？还是有别的反应？

**My answer:**
这个游戏没什么要填的，所以不太涉及。但是要是有人疯狂连按各种键、或者按下不是箭头空格的键，希望游戏不要崩溃、不要弹什么提示框，就静静地忽略掉就行。方块超出边界或者堆到顶上，规规矩矩 Game Over 就好。

## Q6 / 8

**Question:**
> 如果这个工具突然多了一个功能，你反而会觉得『搞这个干嘛，反而把简单的事弄复杂了』——是什么功能？说两三个。

**My answer:**
第一，登录/注册账号——我就摸鱼五分钟还要先注册谁愿意。第二，全球排行榜/社交分享——我就跟自己玩，不想跟陌生人比。第三，皮肤、背景音乐选择、难度档位这种装饰功能——经典版就只有一种速度递增的玩法，多了反而显得啰嗦。

## Q7 / 8

**Question:**
> 用了一个月之后，你怎么判断『这个工具是有用的』？不是『感觉不错』那种——是具体能数出来或看得见的事。比如：『我妈现在不用每周日花两小时算账了。』

**My answer:**
三件事：1) 我书签里固定留着这个网页，每周至少打开两三次；2) 我的最高分从第一次的几百分爬到一万分以上；3) 一个月里玩了几十局，没出现过一次卡死、白屏、或者要刷新页面的情况。

## Q8 / 8

**Question:**
> （最后一题）下面我会把前面听到的复述一遍，请你看看哪里我听错了或听漏了——别客气，挑错就是帮我。如果你想直接补充什么，请在这里写出来。

**My answer:**
基本上对，再确认几点：(1) 整个游戏就是一个 index.html 文件，打开就能玩，不要任何后端、不要 CDN、不要外部资源；(2) 用键盘操作：左右箭头平移、上箭头旋转、下箭头软降、空格开始或重开（不需要硬降）；(3) 经典 NES 七种方块（I O T S Z J L）、10x20 棋盘、消行计分、随等级加速；(4) 最高分用浏览器自带的本地存储记，不要服务器；(5) 一个画面就够了，不要菜单页、设置页、欢迎页。
