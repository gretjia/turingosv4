## Self-critique outcome

PASS — feels academic / austere / intentional, not generic AI v0.dev output.

- No purple gradient on white.
- No Inter / Roboto / Arial as primary face.
- No card-grid SaaS chrome.
- Distinctive typography pair (Fraunces variable serif + JetBrains Mono + IBM Plex Sans).
- Restrained accent palette anchored on oxidized teal `#1F6E6B`.
- Semantic landmarks: `<header>` / `<nav>` / `<main>` / `<footer>` present.

## Iterations during self-check

None — the first render passed the bar. Decisions that pre-emptively
avoided common generic-AI failure modes:
- Picked Fraunces (editorial variable serif) instead of Space Grotesk (overused).
- Picked oxidized teal `#1F6E6B` instead of cobalt/violet (clichéd accent colors).
- Used hairline borders instead of soft drop shadows.
- Used monospace caps + 0.18em letter-spacing for nav/labels instead of regular sans.
- Used italic Fraunces for page titles to signal "editorial" not "marketing".
